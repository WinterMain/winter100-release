默认

浏览器的默认行为等效于：

table {border-collapse: collapse;}
td    {padding: 0px;}

         

细胞填充

设置单元格内容和单元格壁之间的空间量

table {border-collapse: collapse;}
td    {padding: 6px;}

        

单元间距

控制表格单元格之间的空间

table {border-spacing: 2px;}
td    {padding: 0px;}

        

都

table {border-spacing: 2px;}
td    {padding: 6px;}

        

两者（特殊）

table {border-spacing: 8px 2px;}
td    {padding: 6px;}

        

注意：如果已border-spacing设置，则表示border-collapse表的属性为separate。
