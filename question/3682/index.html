我通过在webpack.config中添加缺少的加载器解决了类似的问题。不知道为什么这可以解决问题，但是您可能还想看看是否缺少装载程序。

（请清楚地说-我最初发现问题是在我将库降级到早期版本时出现的加载器问题。Webpack错误消息说缺少json加载器，因此我在配置中添加了以下内容，并且然后就可以回到最新版本：

module:{
loaders : [
  {
      test: /\.json$/,
      loader: "json-loader"
    }
]
}

）
